\subsection{Detailed Description}
Used to hold status messages. 

\hyperlink{classJStatus}{J\-Status} is used all over the framework, and is very useful when a more comprehensive return value is needed by a function. Often just true/false is not enough. With this class you can push a status, then subsequently poll the last status message. \begin{DoxyWarning}{Warning}
The \hyperlink{classJStatus_af4a8242bee3345146577233f7481d388}{J\-Status\-::\-N\-O\-N\-E} constant returns true 
\end{DoxyWarning}
