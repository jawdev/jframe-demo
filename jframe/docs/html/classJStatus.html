Detailed Description

Used to hold status messages.

JStatus is used all over the framework, and is very useful when a more comprehensive return value is needed by a function. Often just true/false is not enough. With this class you can push a status, then subsequently poll the last status message.

Warning
The JStatus::NONE constant returns true
